# Nameless-Websend
Send commands from your website to your Minecraft server using the Websend plugin

## Requirements
- NamelessMC version 2 (from commit [7d5f9fa](https://github.com/NamelessMC/Nameless/commit/7d5f9fa6ae522f96b2cf0ef49decfbe7f1148dff) onwards)
- [Websend](https://dev.bukkit.org/projects/websend)

## Installation
- Upload the contents of the **upload** directory straight into your NamelessMC installation's directory
- Activate the module in the AdminCP -> Modules tab
